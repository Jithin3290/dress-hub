# 🛍️ React eCommerce Platform

A modern, responsive eCommerce web application built with React. Features a full customer shopping experience and an admin dashboard for managing products, users, and orders.

## 🚀 Features

### 🧑‍💼 Customer Side
- 🛒 Add to Cart / Wishlist
- 🔍 Product filtering and search
- 🧾 Order placement with mock checkout
- 🕒 Recently viewed products
- 🔐 User authentication (signup/login)
- 📱 Fully responsive design

### 🛠️ Admin Panel
- 📦 Manage Products (Add/Edit/Delete)
- 👥 Manage Users
- 📊 Dashboard with charts and statistics
- 🧾 View and update orders

## 🛠️ Tech Stack

- **Frontend**: React, React Router, Axios, Recharts, Context API
- **Backend**: JSON server
